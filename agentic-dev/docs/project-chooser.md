# Project Chooser

Pick something you would still care about in six weeks.

## Good Signs

- More than one source file.
- At least one third-party library.
- Enough logic to need testing.
- Clear users or audience.
- Easy to explain as a portfolio piece.

## Better Project Shapes

- A folder audit tool that reports large files, older files, and empty directories.
- A booking or appointment widget that shows availability and the next open slot.
- A local-first app for tracking gym sessions, meals, or habits with useful reports.
- A CSV cleaner or validator for messy real-world data.
- A small game or simulator with rules, state, and scoring.
- A practical tool for a club, class, or family member who would actually use it.

## Usually Weak Choices

- Todo apps.
- One-page brochure sites.
- Straight clones of tutorial projects.
- Projects with no real user, no real rules, and no reason to revisit them.

## Quick Filter

If you can finish it properly in one evening, it is probably too small for this exercise.
